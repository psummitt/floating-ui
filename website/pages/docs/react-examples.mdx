# React Examples

To understand the examples, first make sure you have read
[Interactions](/docs/react#interactions)!

<Notice>
  Interactions are only available for React DOM, not React
  Native.
</Notice>

## Full guides

Learn how to build these common floating UI components, complete
with a reusable component example. These components heavily
leverage the
[React Context API](https://beta.reactjs.org/apis/react/useContext)
to create "compound components", one of the best use cases for
using context.

- [Tooltip](/docs/tooltip)
- [Popover](/docs/popover)
- [Dialog](/docs/dialog)

## Other

complete and may be a bit hard to understand without the full
